# 🚀 ابدأ الآن في 3 خطوات

## الخطوة الأولى: تثبيت المكتبات
```bash
npm install
```

## الخطوة الثانية: إنشاء قاعدة البيانات
```bash
npx prisma migrate dev --name init
```

## الخطوة الثالثة: تشغيل السيرفر
```bash
npm run dev
```
